Degree program B.Sc. Applied Computer Science (B-AI)
Semester Summer semester 2010
Course Databases (DB)
Course type lecture and practical training
Lecturer Prof. Dr. Thomas Thiel-Clemen
Topics
  • DB, DBS, DBMS, ANSI-SPARC Architecture, JDBC, geographic information systems (GIS)
  • Models, ERM, UML (class), relational model
  • DB design, relationships, recursion, security models, access control, prepared statements, procedures and functions
  • Query languages: projection, selection, cartesian product, join, rename
  • SQL: create/drop/alter table, primary/foreign/unique key, select from where group order, cross/natural/inner/outer/left/right join, (inline) view, insert/update/delete, create user/role, grant/revoke
  • Anomalies, functional dependencies, transitive closure, decomposition, 1NF, 2NF, 3NF, transactions, ACID properties
  • Guest lecture by Dr. Kerstin Puschke about NoSQL, CouchDB and REST
Languages SQL, Java, LaTeX, UML
Technologies Oracle DB, GIS, ERM, JDBC, pst-uml
Tools StarUML, Microsoft Visio
IDE Eclipse, Oracle SQL Developer

Examination type written test
